Global spectra deconvolution

gsd(data: object, options: object): Array<object>
Parameters
data (object) Object data with x and y arrays
Name Description
data.x Array<number>? Independent variable
data.y Array<number>? Dependent variable
options (object = {}) Options object
Name Description
options.shape object (default {}) Object that specified the kind of shape to calculate the FWHM instead of width between inflection points. see https://mljs.github.io/peak-shape-generator/#inflectionpointswidthtofwhm
options.sgOptions object? Options object for Savitzky-Golay filter. See https://github.com/mljs/savitzky-golay-generalized#options
options.sgOptions.windowSize number (default 9) points to use in the approximations
options.sgOptions.polynomial number (default 3) degree of the polynomial to use in the approximations
options.minMaxRatio number (default 0.00025) Threshold to determine if a given peak should be considered as a noise
options.broadRatio number (default 0.00) If broadRatio is higher than 0, then all the peaks which second derivative smaller than broadRatio * maxAbsSecondDerivative will be marked with the soft mask equal to true.
options.noiseLevel number (default 0) Noise threshold in spectrum units
options.maxCriteria boolean (default true) Peaks are local maximum(true) or minimum(false)
options.smoothY boolean (default true) Select the peak intensities from a smoothed version of the independent variables
options.realTopDetection boolean (default false) Use a quadratic optimizations with the peak and its 3 closest neighbors to determine the true x,y values of the peak?
options.heightFactor number (default 0) Factor to multiply the calculated height (usually 2)
options.derivativeThreshold number (default -1) Filters based on the amplitude of the first derivative
Returns

joinBroadPeaks

src/post/joinBroadPeaks.js

This function try to join the peaks that seems to belong to a broad signal in a single broad peak.

joinBroadPeaks(peakList: Array, options: object)
Parameters
peakList (Array) A list of initial parameters to be optimized. e.g. coming from a peak picking [{x, y, width}] .
options (object = {}) options
Name Description
options.width number (default 0.25) width limit to join peaks.
options.shape object (default {}) it's specify the kind of shape used to fitting.
options.shape.kind string (default 'gaussian') kind of shape; lorentzian, gaussian and pseudovoigt are supported.
options.optimization object (default {}) it's specify the kind and options of the algorithm use to optimize parameters.
options.optimization.kind string (default 'lm') kind of algorithm. By default it's levenberg-marquardt.
options.optimization.options object (default {}) options for the specific kind of algorithm.

Optimize the position (x), max intensity (y), full width at half maximum (width) and the ratio of gaussian contribution (mu) if it's required. It supports three kind of shapes: gaussian, lorentzian and pseudovoigt
